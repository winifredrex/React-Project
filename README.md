1. What is NPM?

NPM (Node Package Manager) is a tool that comes with Node.js and allows developers to install, update, and manage packages (libraries) needed in JavaScript projects.

It helps you:

Install dependencies like React

Run scripts (npm start, npm run dev)

Manage versions of tools and libraries


2. What is package.json?

package.json is a file created in every Node/React project that stores:

The project name and version

Dependencies (libraries your project uses)

Scripts (like npm run dev)

Project configuration

It acts as the blueprint for your project.

3. What is Vite?

Vite is a fast tool used to create and run React projects.
